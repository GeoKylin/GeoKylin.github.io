安装 Ruby：Linux、macOS 和 Windows

通常，当你在计算机上安装 Ruby 的时候，你所安装的是“Ruby 解释器”，它的功能是理解以 Ruby 编写的程序，并提供丰富的功能扩展和程序库。对于那些某些版本包含而其他版本不包含的额外功能，你可以随时单独安装它们。本文将介绍如何在 Linux、macOS 和 Windows 操作系统上安装 Ruby，当然，作为开源编程语言，许多 Linux 和 macOS 的计算机都自带了 Ruby。

Tip

所谓的“解释”是指程序在运行前不进行编译，而是在运行程序的时候才翻译成机器语言，专门的解释器负责在每个语句执行的时候解释程序代码。这样解释型语言每执行一次就要翻译一次，效率比较低。

无论你在哪个平台下安装 Ruby，在你开始安装 Ruby 前，你可以用下面的命令查看你的计算机中是否已安装 Ruby 并检查它的版本。当然，在你完成安装后，也可以使用这个命令来检查成功与否。

$

ruby -v  #Linux、macOS 和 Windows

Linux 系统下安装 Ruby

Linux 系统下安装 Ruby 最简单的方式是使用 yum 或 apt-get 命令。对于 CentOS、Fedora 或 RHEL 系统，你可以使用 yum 命令，而对于 Debian 或 Ubuntu 系统，你可以使用 apt-get 命令来安装 Ruby。

$
$

sudo yum install ruby	        #CentOS、Fedora、RHEL
sudo apt-get install ruby-full	#Debian、Ubuntu

macOS 系统下安装 Ruby

macOS 10.3 以后的系统都内置了某个版本的 Ruby，如果你要在 macOS 系统下安装 Ruby，最简单的方式是使用 brew 命令。

$

brew install ruby	#macOS

Windows 系统下安装 Ruby

Windows 系统并不内置 Ruby，你可以在 Ruby 官方网站下载 rubyInstaller 来安装 Ruby 环境。

  1. 下载 rubyinstaller：点击这里下载;

  2. 下载 rubyinstaller 之后，解压到新创建的目录下；

  3. 双击 rubyinstaller.exe 文件，启动 Ruby 安装向导；

  4. 点击 Next，记得勾选 Add Ruby executables to your PATH（添加环境变量），继续直到 Ruby 安装程序完成 Ruby 安装为止。
